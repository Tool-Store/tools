## Repository Purpose

This repo contains open-source MCP tool projects for the Tool Store ecosystem. Each tool folder is a separate project with its own language, runtime, dependencies, and Vibe Code documentation.

## Platform Contract

All tools in this repo must include:
- `tool-data.yaml` - MCP contract and marketplace metadata
- Tool-level PROJECT.md, ARCHITECTURE_INDEX.md, PLAN.md (optional but recommended)

## Working with Tools

1. Select the tool folder you want to work on
2. Read its PROJECT.md for capabilities and constraints
3. Check its ARCHITECTURE_INDEX.md for code organization
4. Review/update its PLAN.md for active work
5. Make changes following that tool's local conventions
