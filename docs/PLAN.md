# Repo Index: Tool Plans

This file indexes active execution plans for tools in this repository. Each tool's plan lives inside that tool's folder and drives its development.

## Working with Tool Plans

1. Select the tool you want to work on
2. Read its PLAN.md for current tasks and acceptance criteria
3. Update the tool's PLAN.md as work progresses
4. This repo-level index only needs updates when adding new tools

## Template for New Tool Plans

Create a PLAN.md inside your tool folder with:
- Tool goal
- Current state
- Task backlog (prioritized)
- Definition of done
- Verification commands
